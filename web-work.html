My personal projects for web design/development

Aerom

Being a person who likes the smell and touch of new notebooks, journals, etc. and that's where I found inspiration for Aerom. The website has a modern feel and is minimalistic. Normally I use a sans-serif fonts for headings and serif for paragraph text. This time switching it up and challenging myself to do it differently. I used a font downloaded from the internet called Butler for the headings and Fira Sans for the alternate text. The backgrounds for the header section and footer section challenged me to think outside the box and find a way to create slanted effect, which I did using the power of borders.

Digim Marketing

This website was the second website that I ever made. It makes use of the Bootstrap 3 framework. The agency is a small fictitious marketing company which has only four employees. First, I sketched the idea for the website with wireframes, not highly detailed but it was enough to give myself direction. Second, I started to code the website and added little features with jQuery, such as hover effects and click functions. This website was created with HTML, CSS, SASS and jQuery.

Movie API

The Movie API project is a website that I created where you can enter the name of a movie and it will get the movie information from the website OMDb API and display information about the movie such as the year the movie was released, rating, runtime, actors, plot and more. It uses AJAX to access the site where all the information is stored to display the information.
